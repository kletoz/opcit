NAME
----
opcit - Operador em C Intencionalmente Tosco

SYNOPSIS
--------
'opcit' [--version] COMMAND [args]

DESCRIPTION
-----------
opcit é um operador construído para aprendizado de conceitos fundamentais de C.

OPTIONS
-------
--version::
    Show version of 'opcit' and exit.

COMMANDS
--------

add
~~
Add two integers.
